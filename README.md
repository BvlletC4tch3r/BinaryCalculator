# Binary Calculator
This is an **educational project** that aims to introduce binary arithmetic operations. These operations are performed with **logical operators** in C#, following different types of algorithms. It uses the **WPF framework** to visually display an interactive UI.

Despite using C#, the project has different classes for handling and storing bits.

# Usage

This project was developed in **Visual Studio 2022**. Therefore, you will need a version close to this one.
